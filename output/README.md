This README file lists out the output files from running the HybPiper pipeline.

- `busco1187_sptree.tre`: FastTree species tree in Newick format with updated tip labels
- `busco1187_sptree_cf.tre`: FastTree species tree with concordance vectors as node labels
- `busco1187_cf_all.tsv`: summary table of gCF and sCF values across all branches
- `busco1187_cf_major.tsv`: summary table of gCF and sCF values across major groupings (i.e., genus, subgenus, section)

> **Note**: FastTree species tree was inferred under GTR model
