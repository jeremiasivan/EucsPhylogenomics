# EucsPhylogenomics

This repository stores the codes and files used to perform phylogenomic analyses on 837 Eucalypts from Currency Creek (801 samples), <a href="https://doi.org/10.1101/gr.277999.123">Ferguson et al.</a> (35 samples), and meelup-mallee (1 sample).
